# Guia rapida de uso

1) Inicia sesion con el usuario demo (ver README).
2) Dashboard: revisa KPI y alertas.
3) Registra un animal y asigna establecimiento.
4) Crea un producto, luego un lote con stock.
5) Registra un tratamiento y una aplicacion usando ese lote.
6) Verifica el retiro activo en reportes.
7) Prueba bloqueos creando un movimiento a faena.

Tips
- Usa el buscador global para arete, animal o lote.
- Exporta tablas a CSV desde el menu de acciones.
